ITAhM

ITAhM은 Infrastructure Topology Administration Module (기업 IT 시스템 토폴로지 관리 모듈)의 약어이며, high performance를 의미 하는 h가 더해진 서비스의 이름 입니다. 한글로는 "아이탐" 입니다.

향후 확장 가능한 더 큰 프로젝트의 일부 Module로써 개발되었습니다.

ITAhM Demo 미리 보기!

구성

  • 에이전트: 자바 응용프로그램으로 다음과 같이 구성되어 있습니다.
    • snmp 서비스
    • database
    • http 서버

    에이전트는 하나 이상 설치할 수 있으며 이렇게 함으로써 다수의 그룹을 만들 수 있습니다.

  • 사용자 인터페이스: 웹브라우저

    http://itahm.com

특징

  • High Performance(고성능)

    불과 500KBytes 크기의 자바 응용프로그램으로, 일반 개인용 desktop 수준으로 충분히 많은 수의 장치들을 관리할 수 있습니다.

  • UX(User Experience)

    사용자의 최소 조작으로 모든 기능을 사용할 수 있도록 사용자 경험이 최대한 반영되어 있습니다.

  • Compact(최적화)

    에이전트에는 직접 개발한 웹서버, 데이터베이스, snmp서비스가 모두 포함되어 있으며,
    각 서비스들은 에이전트를 구성하는 기능만을 포함하고 있어 매우 정밀하고 최적화 되어 있습니다.

  • technologies(기술)

    직접 개발한 에이전트 서비스 이외에 html5 표준을 따르는 고성능의 graphic 라이브러리등 앞선 기술을 적용하고 있습니다.

기술

아래 나열하는 모든 기술들은 기존 엔터프라이즈 솔루션이 가지고 있지 않은 기술들입니다.

  • 확장성

    에이전트는 java 응용프로그램으로 자바를 지원하는 windows나 linux등 모든 os에 설치 가능 합니다.

    Uer Interface는 웹브라우저를 통해 구현하였고 Internet Explorer, google Chrome, Firefox, Opera등 모든 브라우저에서 동작합니다.

  • 통신

    웹브라우저와 웹서버 그리고 3rd party 서비스간 json(JavaScript Object Notation) 통신 기술을 보유하고 있습니다.

  • 그래픽

    Topology는 브라우저의 성능에 비례하여 거의 무한히 많은 장치들을 그려낼 수 있고 확대/축소 이동시 지연현상을 느끼지 못할만큼 빠릅니다.

    Chart는 1개월간 요약 데이터를 드래그에 의해 간편하게 검색 할 수 있으며 1분단위 세부정보와 실시간 모니터링을 제공합니다.

  • 모바일

    에이전트에서 발생하는 알림을 모바일 앱으로 수신하도록 하여 기존 sms(단문서비스)를 대체합니다.

장점(요약)

  • 비용

    기본기능을 무료로 제공하고 있습니다.

    알림에 사용되는 sms문자 비용을 절감할 수 있습니다.

  • 성능

    에이전트 서버는 저사양 서버 또는 데스크탑에서 충분히 가동할 수 있을 만큼 자원대비 성능이 탁월하며
    여러대의 서버를 병렬로 구성하여 그룹화 할수도 있습니다.

  • 기능과 편의(UX)

    기존 엔터프라이즈 솔루션이 제공하는 거의 모든 기능을 아이탐만의 방식으로 편리하고 최적의 서비스를 제공합니다.

에이전트 설치

  • ITAhM agent를 설치할 폴더를 준비(생성) 합니다.

    예) "c:\Users\홍길동\itahm" (windows)

  • 최신의 아이탐 에이전트를 다운로드 하여 위에서 준비한 폴더에 위치합니다.

  • ITAhM.jar 파일을 실행시킵니다. 윈도우즈의 경우 command 콘솔에서, 리눅스의 경우 백그라운드로 실행합니다.

    java -jar ITAhM.jar (windows)

    nohup java -jar ITAhM.jar & (linux)

  • 홈페이지 상단의 Connect 버튼을 눌러 접속합니다.

유의사항

  • 모든 브라우저에서 테스트 되었으나, 일부 브라우저에서 성능이 더 안좋을 수 있습니다.

  • 일반적으로 크롬 브라우저에서 양호한 성능을 보입니다.

문의사항은 itahm2014@gamil.com 으로 보내주십시오. 감사합니다.

Map

아이탐은 무한하게 확장 가능한 구성도를 제공합니다. 장비의 타입에 따라 아이콘 모양을 달리하며, 기본적으로 제공되는 아이콘 이외 사용자 정의 아이콘을 사용할 수도 있습니다.

Scroll to zoom in/out.

Drag to move map or device.

Click device for something.

  • 마우스를 스크롤하여 간편하게 맵을 확대하거나 축소합니다.
  • 화면을 드래그하여 구성도 전체를 이동하고 장비를 선택한 후 드래그 하면 해당 장비를 이동할 수 있습니다.
  • 장비를 선택하고
    • 장비의 종류, snmp설정, 이름등 기본 정보를 조회하거나 수정합니다.
    • 닻 모양의 앵커를 드래그 하여 다른 장비와 연결을 설정합니다.
    • 차트 모양의 아이콘을 클릭하여 장비의 성능을 조회합니다.
  • 장애 상태이거나 임계를 초과한 장비는 붉은 원을 점멸하여 표시해 줍니다.

성능

응답시간, CPU 사용률, 디스크 및 메모리 사용량 , interface 송수신량 등 장비의 5가지 상태를 한번에 확인할 수 있습니다.

  • 각각의 정보를 클릭하여 최근 한달간 성능의 추이를 조회할 수 있습니다.

차트

Drag to move before or later.

Click a lock button to fix a date.

  • 차트 하단 커서를 드래그하여 다른 날짜를 검색할 수 있습니다.
  • 두 커서는 검색 시작날짜와 검색 종료날짜를 의미 합니다.
  • 총 한달 간의 자료를 1시간 최대/평균/최소값으로 보여줍니다.
  • 반투명한 영역의 윗쪽이 최대값, 아랫쪽이 최소값이며 그 사이의 굵은 선이 평균값 입니다.
  • 날짜 단위로 검색할 수 있고 차트의 공간이 충분할 경우 1분간격의 최대값을 볼 수도 있습니다.

안드로이드 앱

  • 아이탐에서 발생하는 이벤트는 안드로이드 앱으로 전송됩니다.
  • 또한 구성원들이 직접 원하는 메세지를 앱으로 전송할 수도 있습니다.

Installation

  • Create directory for ITAhM agent

    e.g. "c:\Users\ABC\itahm" (windows)

  • Download latest ITAhM.1.1.3.15.jar into above directory

  • execute ITAhM.jar

    java -jar ITAhM.jar (windows)
    nohup java -jar ITAhM.jar & (linux)

  • Connect!

  • ITAhM agent를 설치할 폴더를 준비(생성) 합니다.

    예) "c:\Users\홍길동\itahm" (windows)

  • 최신의 아이탐 에이전트( ITAhM.1.1.3.15.jar )를 다운로드 하여 위에서 준비한 폴더에 위치합니다.

  • ITAhM.jar 파일을 실행시킵니다. 윈도우즈의 경우 command 콘솔에서, 리눅스의 경우 백그라운드로 실행합니다.

    java -jar ITAhM.jar (windows)
    nohup java -jar ITAhM.jar & (linux)

  • 홈페이지 상단의 Connect 버튼을 눌러 접속합니다.

Browser compatibility

  • Chrome, Firefox, Opera, Internet explorer 11 에서 테스트 되었으며, 현재 Chrome 브라우저에서 가장 안정적이고 좋은 성능을 보이고 있습니다.
  • Internet Explorer 10 이하에서는 오작동 할 수 있습니다.
