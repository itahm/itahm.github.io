Installation

  • Create directory for ITAhM agent

    e.g. "c:\Users\ABC\itahm" (windows)

  • Download latest ITAhM.1.1.3.13.jar into above directory

  • execute ITAhM.jar

    java -jar ITAhM.jar (windows)
    nohup java -jar ITAhM.jar & (linux)

  • Connect!

  • ITAhM agent를 설치할 폴더를 준비(생성) 합니다.

    예) "c:\Users\홍길동\itahm" (windows)

  • 최신의 아이탐 에이전트( ITAhM.1.1.3.13.jar )를 다운로드 하여 위에서 준비한 폴더에 위치합니다.

  • ITAhM.jar 파일을 실행시킵니다. 윈도우즈의 경우 command 콘솔에서, 리눅스의 경우 백그라운드로 실행합니다.

    java -jar ITAhM.jar (windows)
    nohup java -jar ITAhM.jar & (linux)

  • 홈페이지 상단의 Connect 버튼을 눌러 접속합니다.

Browser compatibility

  • Chrome, Firefox, Opera, Internet explorer 11 에서 테스트 되었으며, 현재 Chrome 브라우저에서 가장 안정적이고 좋은 성능을 보이고 있습니다.
  • Internet Explorer 10 이하에서는 오작동 할 수 있습니다.
